io.js_anonymous Class

Defined in: src\js\io.js:6
Module: io.js

The class that holds the variables and functions created in the namespace of this file. The constrcutor of this object is only called once, because it is an anonymous function.

The functions defined in this class are later added to the MARKUS global object and can from that point on be called as markus.io.FUNCTION_NAME
